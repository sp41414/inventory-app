# Inventory App

## Features:

1. Add products
2. Filter products by genre
3. Delete products
4. Admin authentication by password to delete and add products

## Steps:

1. Run `npm i`
2. Create a new postgres database
3. Move `.env.template` to `.env`
4. Change the values in `.env`
5. Run `npm run db`
6. Run `npm run dev`
7. Go to https://localhost:{PORT-from-.env}
